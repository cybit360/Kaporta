# Kaporta Group Website

Professional construction & logistics website for **Kaporta Investment Sierra Leone Limited**, operating in West Africa with headquarters in Sierra Leone. Built with Next.js 14+, TypeScript, and Tailwind CSS v4.

## Getting Started

```bash
npm install
npm run dev     # Start dev server on http://localhost:3000
npm run build   # Production build
npm run lint    # Run ESLint
```

## Tech Stack

- **Framework:** Next.js 14+ (App Router) with TypeScript
- **Styling:** Tailwind CSS v4 with custom design tokens
- **Fonts:** Poppins (headings), Inter (body), JetBrains Mono (code)
- **Icons:** lucide-react

## Site Structure

| Page | Route | Description |
|------|-------|-------------|
| Home | `/` | Hero, services, projects, testimonials, CTA |
| About Us | `/about` | Company history, mission, vision, values |
| Leadership | `/about/leadership` | Team member profiles |
| Certifications | `/about/certifications` | Certifications & partnerships |
| Services | `/services` | All 6 service domains |
| Service Detail | `/services/[slug]` | Individual service pages |
| Projects | `/projects` | Portfolio with category filters |
| Careers | `/careers` | Job listings |
| Internships | `/careers/internships` | Training programs |
| Testimonials | `/testimonials` | Client reviews |
| Blog | `/blog` | Articles & insights |
| Blog Post | `/blog/[slug]` | Individual articles |
| Request a Quote | `/quote` | RFQ form |
| Contact | `/contact` | Contact form & info |
| Payment | `/payment` | Payment information |
| Privacy | `/privacy` | Privacy policy |
| Terms | `/terms` | Terms of use |
| Accessibility | `/accessibility` | Accessibility statement |
